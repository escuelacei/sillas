# Sillas

Una pequeña página web estática creada con HTML y CSS que sirve como prototipo o catálogo sencillo para mostrar diseños y estilos de sillas. Este repositorio contiene un ejemplo mínimo que puedes usar como base para practicar HTML/CSS o para empezar un proyecto frontend ligero.

**Estado:** Proyecto inicial (plantilla estática).

**Tecnologías:**
- HTML 5
- CSS 3

**Estructura del proyecto (resumen):**
- `index.html`: Página principal del proyecto.
- `css/index.css`: Estilos globales del proyecto.

**Objetivo:**
- Proveer una plantilla simple y modular para crear una página de catálogo de sillas.
- Servir como ejemplo de estructura de archivos y estilos básicos.

Cómo ejecutar localmente
- Opción rápida: abrir `index.html` en tu navegador (doble clic o `open index.html` en macOS).
- Opción recomendada (servidor local, evita problemas con rutas):
```
cd /Users/eduardofierro/Desktop/sillas
python3 -m http.server 8000
# luego abre http://localhost:8000 en el navegador
```
- Si usas VS Code, la extensión "Live Server" sirve muy bien para desarrollo en vivo.

Cómo contribuir
- Haz un fork del repositorio.
- Crea una rama nueva con el nombre `feat/descripcion-corta` o `fix/descripcion-corta`.
- Haz tus cambios y sube la rama.
- Abre un Pull Request describiendo tus cambios.
